# Numblers

## Quickstart

1. Clone this repo.
2. Change into the project directory.
3. Install dependencies with `npm install`.
4. Copy `.env.example` to `.env`.
5. Add your Supabase URL and anon key to `.env`.
6. Run the dev server with `npm run dev`.
7. Open the printed URL in your browser.
8. Sign in with GitHub to start using the app.
9. Add Supabase SQL for puzzles and attempts later in the Supabase dashboard.
10. Hack on Numblers!
